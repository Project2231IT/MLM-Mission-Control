## NVIDIA GPU Driver Install

- **GPU:** NVIDIA GeForce RTX 4060 (AD107, Dell subsystem, 8GB VRAM)
- **Also present:** Intel integrated display controller
- Was running on **nouveau** open-source driver (no CUDA)
- Hit multiple issues during install:
  - Old nvidia-535 driver remnants causing dpkg conflicts
  - `libnvidia-cfg.so.1` file conflict — resolved with `dpkg --force-overwrite`
  - DKMS build failures on old kernel 6.8.0-101 and current 6.8.0-106
  - Root cause: GCC 11 (Ubuntu 22.04 default) missing `-ftrivial-auto-var-init=zero` flag required by nvidia-580
  - **Fix:** Installed gcc-12, set as default via update-alternatives, then DKMS built successfully
- **Final driver:** nvidia-driver-580-open (v580.126.20), CUDA 13.0
- Verified post-reboot: `nvidia-smi` shows RTX 4060, 8188MiB VRAM, 29°C
- OpenClaw gateway survived reboot cleanly (systemd user service, enabled)
